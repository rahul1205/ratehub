# ratehub

Usage
-------------------

**Requires Docker and Docker Compose.**

```
git clone https://github.com/rahul1205/ratehub.git
```
Download the json in root directory

**Start MySQL**

```
docker-compose up -d
```
**Install requrired dependencies**

```
pip install -r requirements.txt
```
**Run the script**
```
python load_sql.py
```
